
# 🛒 Shop Management System (Java Swing)

A desktop-based **Shop Management System** developed using **Java (Swing GUI)** to manage day-to-day operations of a retail business. This application supports customer handling, employee management, billing, and inventory-related processes.



## 📌 Features

* Customer Registration & Management
* Company/Supplier Registration
* Employee Management
* Address Management
* Invoice Generation
* GRN (Goods Received Note) Handling
* Dashboard/Home Interface
* User-friendly GUI using Java Swing



## 🛠️ Technologies Used

* **Java SE**
* **Java Swing (GUI)**
* **NetBeans IDE**
* **JasperReports** (for reports generation)
